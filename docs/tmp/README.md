# docs/tmp

`bpfopt_design_v3.md` is the only authoritative bpfopt-suite design document.

Other files in this directory are temporary research notes, reviews, benchmark
run summaries, and historical records. They are not canonical architecture
specifications unless they explicitly defer to `bpfopt_design_v3.md`.

`archived/` contains obsolete design drafts and reports superseded by the v3
design, especially older v1/v2 daemon-centric bpfopt/runner documents.
